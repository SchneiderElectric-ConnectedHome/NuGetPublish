# NuGetPublish
Powershell Script for packing and publishing projects to NuGet

###Parameters
- (mandatory) **ProjectFile** *-p* 
  - ex:   *My.Project.csproj*
- (mandatory) **ProjectName** *-n* 
  - ex:   *My.Project*
- **NugetSource** *-n* 
  - default - *"https://www.nuget.org/api/v2/"*
- **ApiKey** *-k* 
  - default - *""*
- **BuildConfiguration** *-c* 
  - default - *"Release"*
